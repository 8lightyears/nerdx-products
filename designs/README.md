# Designs

One folder per concrete product concept: `designs/<slug>/`.

```
designs/<slug>/
  brief.md     idea, copy/slogan, visual direction, references, license, sources, status
  assets/      generated/found artwork (commit small review samples only; keep full-res out of the repo)
  generate/    a script or an AI prompt that produces the visual, when automatable
```

A concept is "production-ready" when `brief.md` gives a designer (or a script)
everything needed to make the final artwork, and every referenced asset has a
recorded license.
