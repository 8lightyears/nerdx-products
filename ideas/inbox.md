# Idea inbox

Raw idea queue for nerdx-products. The owner — or an agent feeling creative —
appends ideas here. Agents pull from the **top**, turn an idea into a
`themes/<theme>.md` analysis and/or a `designs/<slug>/` concept, then strike it
from this list and log it in `docs/done.md`.

Mark agent-proposed additions with `(agent-proposed)`. Keep each entry short — a
line or two; the detail belongs in the theme/design files.
