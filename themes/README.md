# Themes

One file per theme group: `<theme>.md`. Each captures the family's scope, why it
fits the shop's spirit, its sub-ideas, sourced curiosities (URL + date checked),
and notes on how each idea could be visualised or automated.

See `docs/vision.md` for the initial map of theme groups, and `ideas/inbox.md`
for the live queue.
